# Implementation Notes: 024

**Resolve instructions**:
```python
exec_instructions = arguments.get("exec_instructions") or os.environ.get("IDEATE_EXEC_INSTRUCTIONS", "")
```
A falsy value (empty string, None) means no injection.

**Prompt augmentation** (after prompt size validation, before building the cmd):
```python
effective_prompt = prompt
if exec_instructions:
    effective_prompt = (
        f"[EXECUTION INSTRUCTIONS]\n{exec_instructions}\n[END EXECUTION INSTRUCTIONS]\n\n{prompt}"
    )
```
Use `effective_prompt` in the `cmd` array. Use `prompt` (original) for `prompt_bytes` logging.

**Env propagation**:
```python
if exec_instructions:
    env["IDEATE_EXEC_INSTRUCTIONS"] = exec_instructions
```
Add this to the env dict construction block alongside `IDEATE_SPAWN_DEPTH` and `IDEATE_TEAM_NAME`.

**Tool schema parameter placement**: Add `exec_instructions` after `team_name` in `inputSchema.properties`.

**Prompt size check**: The existing validation `if prompt_byte_len > MAX_PROMPT_BYTES` uses `len(prompt.encode("utf-8"))` where `prompt` is the raw argument. This must not change — `effective_prompt` is only built after the validation passes.
